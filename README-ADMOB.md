| البيان | القيمة |
|--------|--------|

## كيفية تغيير معرفات AdMob التجريبية بمعرفاتك الحقيقية

### الخطوة 1: إنشاء حساب AdMob
1. ادخل على https://admob.google.com
2. سجّل حساب جديد أو سجّل دخول
3. أضف تطبيق جديد → اختر Android
4. أنشئ وحدتين إعلانيتين:
   - **Banner** (إعلان شريطي)
   - **Interstitial** (إعلان بيني)

### الخطوة 2: تغيير المعرفات في index.html

افتح ملف `index.html` وابحث عن الأماكن التالية:

#### 1. معرف حساب AdMob (السطر 17):
```
غيّر: ca-app-pub-3940256099942544
إلى: ca-app-pub-XXXXXXXXXXXXXXXX (معرف حسابك)
```

#### 2. معرف إعلان Banner (السطر ~1050):
```
غيّر: data-ad-slot="6300978111"
إلى: data-ad-slot="معرف_إعلان_البانر_حقك"
```

#### 3. معرف إعلان Interstitial (السطر ~1058 و ~1074):
```
غيّر: ca-app-pub-3940256099942544/1033173712
إلى: ca-app-pub-XXXXXXXXXXXXXXXX/XXXXXXXXXX (معرفك الحقيقي)
```

## متى تظهر الإعلانات؟

| نوع الإعلان | متى يظهر | المدة |
|-------------|-----------|-------|
| **Banner** | دائماً أسفل الشاشة عند فتح التطبيق | مستمر |
| **Interstitial** | عند إكمال كل ختمة (100 تسبيحة) | 5 ثواني ثم يُغلق تلقائياً |

## خطوات رفع على GitHub Pages

1. أنشئ Repository جديد Public على GitHub
2. ارفع الملفات: `index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`
3. اذهب إلى Settings > Pages
4. اختر Branch: main > / (root) > Save
5. انتظر دقيقة وبيظهر الرابط

---

## خطوات PWA Builder

1. ادخل https://www.pwabuilder.com
2. الصق رابط GitHub Pages
3. اضغط Start > Package for stores > Android
4. الإعدادات:
   - **Package Name:** `com.tasbih.app`
   - **App Name:** المسبحة الإلكترونية للصلاة
   - **Icon:** ارفع icon-512.png
   - **Signing Key:** استخدم ملف tasbih-app.keystore المرفق
5. حمّل الـ APK

---

## النشر على APKPure

1. سجّل على https://apkpure.com/submit
2. ارفع ملف APK
3. أضف:
   - اسم التطبيق: المسبحة الإلكترونية للصلاة
   - وصف قصير ووصف كامل
   - لقطات شاشة (3-5 صور)
   - أيقونة 512×512
   - تصنيف: Lifestyle / Tools
4. أضف سياسة خصوصية (مطلوبة)
5. اضغط Submit وانتظر المراجعة (1-3 أيام)

## ملاحظات مهمة

- المعرفات التجريبية تعرض إعلانات وهمية للاختبار فقط
- لازم تبدّلها بمعرفاتك الحقيقية قبل النشر عشان تربح
- Google ممكن يرفض التطبيق إذا فيه إعلانات كثيرة أو مزعجة
- الإعلان البيني يظهر مرة وحدة فقط عند كل ختمة (مو مزعج)
